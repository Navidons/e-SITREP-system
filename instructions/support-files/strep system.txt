Data received from over 60 border posts is compiled in the above format and all disminated into one situation report (strep) in a tabule form.

The data from all daily situation reports is used to produce a comprehensive weekly situation report as excel document with all stations as rows and days of week as columns.

This is usually done manually, I want to create an innovative system to automate these daily and weekly reports ie officers at stations can be entering this data from stations then the system is able to generated a daily situation report and weekly situation report.
